# C0rtexCard
A fast, local-first flashcard app built for the way I actually revise — not the way study apps assume everyone revises.

Dark, terminal-inspired UI — DM Mono for data and structure, Bricolage Grotesque for headings
https://ashleyjoao.github.io/C0rtexcard/
## Features

- **Folders & decks** — organise revision by subject with custom emoji/colour tags
- **Four study modes** — flashcards, multiple-choice quiz, memory-match, and a Quizlet-style Test (mixed MC/written/true-false, graded on submit)
- **Weak-card resurfacing** — tracks every rating per card and surfaces what you keep getting wrong
- **Progress dashboard** — mastery rings, streaks, and stats from local history
- **Custom Korean keyboard** — a Hangul composer built from scratch, no IME library
- **Zero setup** — runs entirely on `localStorage`. No sign-up, no server, no data leaves your machine
